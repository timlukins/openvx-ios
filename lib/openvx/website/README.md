InVX.io Website
===============

Also hosted on github (but only the contents of this directory are visible publically).


See:

	https://help.github.com/articles/using-jekyll-with-pages/

Use:

	bundle exec jekyll serve

In this directory to preview the site at http://localhost:4000

When ready, push it to the *public* gh-pages orphan branch by going up one level and doing:

	git subtree push --prefix website origin gh-pages
